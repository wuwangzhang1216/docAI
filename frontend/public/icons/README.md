# PWA Icons

Please add the following icon sizes for PWA support:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can generate these icons from a single high-resolution image (512x512 or larger) using tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

The icons should be square PNG images with transparent backgrounds.
